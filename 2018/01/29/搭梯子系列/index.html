前提

申请云服务需要绑定自己的信用卡只能V卡和M卡，JCB的不知道是否支持。LZ是使用的招商银行的信用卡。卡是否支持Visa看信用卡正面的图标 ps: 申请过程中会扣1美元，但是之后会返回来。

步骤

申请自己的谷歌账号

登录 https://cloud.google.com/ 进入控制台

Compute Engine ->VM实例->创建实例

创建实例注意:

创建成功后:

创建成功后，连接SSH服务器。这里我直接使用浏览器连接（也可以使用其他方式连接）

连接成功后，开始执行以下命令(有些慢，耐心点):

  1. sudo -i
  2. wget -N –no-check-certificate https://raw.githubusercontent.com/FunctionClub/YankeeBBR/master/bbr.sh && bash bbr.sh install
    执行完成后出现如下界面:

Tab切换到No上，回车。
然后出现如下界面，输入Y 回车。

关闭当前窗口，重新连接。

依次执行以下命令:

  1. sudo -i
  2. bash bbr.sh start
  3. wget –no-check-certificate https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocksR.sh && chmod +x shadowsocksR.sh
    ./shadowsocksR.sh

进入以下界面:

除端口和密码需要设置外(端口我设置的是6666)，其他直接回车。
